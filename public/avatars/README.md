# Avatars assistants UWI

Placez ici les images des assistants (portrait, épaules + visage).  
Les chemins sont définis dans `src/assistants.config.js` : `/avatars/uwi-avatar-[id].png`

- `uwi-avatar-sophie.png`
- `uwi-avatar-laura.png`
- `uwi-avatar-emma.png`
- `uwi-avatar-julie.png`
- `uwi-avatar-clara.png`
- `uwi-avatar-hugo.png`
- `uwi-avatar-julien.png`
- `uwi-avatar-nicolas.png`
- `uwi-avatar-alexandre.png`
- `uwi-avatar-thomas.png`

Format recommandé : portrait, hauteur utile ~180px (object-fit: cover), ratio type 3/4.

En l’absence d’image, un placeholder avec l’initiale du prénom s’affiche.
