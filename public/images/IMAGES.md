## Notes techniques

### Fallback
- Toutes les images ont un fallback intégré en cas d'erreur de chargement
- Les sections affichent un gradient de couleur si l'image n'est pas disponible

### Optimisation
- Les images sont chargées avec `loading="lazy"` pour améliorer les performances
- Format recommandé: JPG pour les photos réelles
- Compression recommandée: 80-85% qualité pour équilibrer poids/qualité

### Placement
- Toutes les images doivent être placées dans `/public/images/`
- Le chemin utilisé dans le code est `/images/nom-du-fichier.jpg`

## Comment ajouter les images

1. Placez vos images dans le dossier `public/images/`
2. Assurez-vous que les noms correspondent exactement à ceux listés ci-dessus
3. Les images seront automatiquement affichées sur la landing page

## Alternative: Utiliser des placeholders

Si vous n'avez pas encore les images finales, vous pouvez :
- Utiliser des images de stock similaires
- Créer des mockups avec Figma/Canva
