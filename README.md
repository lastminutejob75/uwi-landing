# UWi Landing Page

Landing page React pour UWi - Agent d'accueil IA multicanal.

## Installation

```bash
cd landing
npm install
```

## Développement

```bash
npm run dev
```

Le serveur de développement démarre sur `http://localhost:3000`

## Build

```bash
npm run build
```

Les fichiers de production sont générés dans `frontend/landing/`

## Technologies

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React (icônes)
